## 1.0.0 - 2026-01-28

Initial release of mcp-popup-ui.

### Added

- `ask_user` tool for single-choice selection (radio buttons)
- `ask_user_multiple` tool for multi-selection (checkboxes)
- Browser-based popup UI with keyboard navigation
- Option descriptions with Markdown formatting
- Recommended option highlighting
- Custom "Other" input option
- Optional comments field for additional user feedback
- Skip button to bypass questions
- Explanation request feature for options
- VS Code, Claude Desktop, and LM Studio integration
